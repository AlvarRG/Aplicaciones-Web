Bistro FDI

Bienvenido a la gestión inteligente

Bistro FDI es una aplicación web integral diseñada para transformar la experiencia en el restaurante. Nuestra plataforma permite a los clientes realizar pedidos personalizados desde sus dispositivos, mientras que el personal de cocina y sala optimiza la preparación y entrega en tiempo real. Desde la gestión de inventario por parte del gerente hasta el sistema de recompensas para clientes habituales, Bistro FDI digitaliza cada paso para ofrecer un servicio rápido, eficiente y moderno.

Esta página sirve como documentación inicial del proyecto, detallando su arquitectura, diseño de interfaz y el plan de desarrollo para el curso 2025/2026.
